class fluent.scheduler.Scheduler(config, input, processor, output, task_name=None, debug=False)[源代码]¶

常驻进程或定时任务

不在同一个进程应在外部实现，不改变同一个进程内部逻辑

参数:
  • config – 配置
  • input (Inputer) – job.input
  • processor (Processor) – job.Processor
  • output (Outputer) – job.Outputer
  • job_dir – job所在路径
